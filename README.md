# Connectify

A real-time video conferencing and collaboration platform built using React, Node.js, Socket.io and WebRTC.

## Features

- User Authentication
- Create & Join Meeting Rooms
- Real-time Video Calling
- Live Chat
- Screen Sharing
- Collaborative Whiteboard
- Participants Panel
- Meeting Timer
- Copy Room Code
- Modern Dark UI
- Responsive Design

## Tech Stack

### Frontend
- React.js
- Tailwind CSS
- Socket.io Client
- Simple Peer

### Backend
- Node.js
- Express.js
- Socket.io
- MySQL
